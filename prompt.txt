## PART V: RISK MANAGEMENT — ZERO COMPROMISE FRAMEWORK

### 5.1 Capital Allocation & Position Sizing

```python
class PositionSizingEngine:
    """
    ATR-based volatility-adjusted position sizing.
    Hard rule: Never risk more than 1% of total capital per trade.
    """
    
    MAX_RISK_PER_TRADE_PCT = 0.01           # 1% of total portfolio
    MAX_SINGLE_STOCK_EXPOSURE = 0.08        # 8% max in one stock
    MAX_SECTOR_EXPOSURE = 0.25              # 25% max in one sector  
    MAX_CORRELATED_EXPOSURE = 0.40          # 40% max in high-correlation basket
    MAX_CONCURRENT_TRADES = 10              # Intraday; 6-8 for swing
    MIN_RISK_REWARD = 2.0                   # 1:2 minimum; target 1:3
    CASH_BUFFER_MIN = 0.15                  # Always keep 15% cash minimum
    
    def compute_position_size(self, capital, entry_price, stop_loss, atr):
        risk_amount = capital * self.MAX_RISK_PER_TRADE_PCT
        stop_distance = abs(entry_price - stop_loss)
        
        # ATR adjustment: wider stop = smaller position
        atr_adjusted_stop = max(stop_distance, 0.5 * atr)
        
        raw_qty = risk_amount / atr_adjusted_stop
        max_qty_by_exposure = (capital * self.MAX_SINGLE_STOCK_EXPOSURE) / entry_price
        
        final_qty = min(raw_qty, max_qty_by_exposure)
        
        # Round down to lot size for F&O; to 1 for equities
        return floor(final_qty)
    
    def kelly_fraction(self, win_rate, avg_win, avg_loss):
        """Kelly Criterion — use half-Kelly for safety"""
        full_kelly = win_rate - ((1 - win_rate) / (avg_win / avg_loss))
        return full_kelly * 0.5  # Half-Kelly always
```

```python
class ExecutionEngine:
    
    SUPPORTED_ORDER_TYPES = [
        "MARKET", "LIMIT", "SL",       # Stop-loss market
        "SL-M", "CNC",                  # Cash & Carry (delivery)
        "MIS", "NRML"                   # Margin intraday, F&O normal
    ]
    
    def place_order(self, signal):
        """
        Full execution pipeline with safety checks.
        Never executes without pre-checks passing.
        """
        # Pre-execution safety checks
        assert signal.confidence >= 65, "Confidence below threshold"
        assert signal.rr_ratio >= 2.0, "RR below minimum"
        assert self.portfolio.available_capital > signal.required_capital
        assert self.risk_manager.can_add_position(signal)
        assert not self.is_trading_halted()
        assert self.market_is_open()
        
        # Order construction
        order = {
            "symbol": signal.symbol,
            "exchange": signal.exchange,      # NSE or BSE
            "transaction_type": signal.side,   # BUY or SELL
            "quantity": signal.position_size,
            "order_type": self.select_order_type(signal),
            "price": signal.entry_price,
            "trigger_price": signal.stop_loss,
            "product": "MIS" if signal.intraday else "CNC",
            "validity": "DAY",
            "disclosed_quantity": 0,           # Hidden order for large positions
            "tag": f"APEX_{signal.strategy_id}_{signal.signal_id}"
        }
        
        # Bracket order for automated SL and target
        bracket = {
            "stoploss": signal.stop_loss,
            "target": signal.target,
            "trailing_stoploss": signal.trail_amount
        }
        
        # Execute with retry logic (3 attempts, exponential backoff)
        result = self._execute_with_retry(order, bracket, max_retries=3)
        
        # Post-execution logging and monitoring
        self._log_execution(signal, result)
        self._start_position_monitor(signal.symbol, signal)
        
        return result
    
    def smart_execution(self, signal):
        """
        Minimize market impact using TWAP/VWAP execution for large orders.
        """
        if signal.position_size * signal.entry_price > 500000:  # >5 lakh
            return self.twap_execution(signal, intervals=5)
        else:
            return self.place_order(signal)
```

### 7.2 Position Monitoring & Dynamic Management

```python
class PositionMonitor:
    """Runs every 60 seconds during market hours"""
    
    def monitor_all_positions(self):
        for position in self.open_positions:
            current_price = self.get_live_price(position.symbol)
            
            # Update trailing stop
            if position.is_long and current_price > position.highest_seen:
                position.highest_seen = current_price
                position.trailing_stop = current_price - 2 * position.atr
                self._update_stop_order(position)
            
            # Check exit conditions
            if self.should_exit_early(position, current_price):
                self.exit_position(position, reason="Early exit signal")
            
            # Profit scaling: take partial profits at 1:1 RR
            if position.pnl_ratio >= 1.0 and not position.partial_exit_done:
                self.partial_exit(position, fraction=0.33)
                position.partial_exit_done = True
                position.stop_loss = position.entry_price  # Move to BE
    
    def should_exit_early(self, position, current_price):
        return any([
            current_price <= position.stop_loss,           # Stop hit
            self.new_bearish_signal_emerged(position),     # Counter signal
            self.market_circuit_triggered(),               # Market halt
            self.correlation_risk_exceeded(),              # Portfolio risk
            self.time_stop_triggered(position),            # Time stop
        ])
```

## PART XI: IRON-CLAD CONSTRAINTS & ETHICAL OPERATING RULES

The following rules are non-negotiable and hardcoded into every layer of the system:

1. **Capital Preservation First:** A 15% drawdown triggers mandatory system halt and strategy review. Protect the principal above all else.

2. **No Gambling:** Every trade must have a documented thesis, predefined stop, and predefined target BEFORE entry. No "see how it goes" positions.

3. **No FOMO Trades:** If the optimal entry is missed by more than 2 ATR, the trade is abandoned. Chasing is strictly prohibited.

4. **Regulatory Compliance:** System must respect all SEBI regulations — no circuit limit stocks, no F&O ban period stocks, no insider information inputs.

5. **Explainability:** Every signal must have a complete, human-readable reasoning summary. Black-box outputs are not accepted.

6. **Backtest Before Live:** No strategy goes live without passing the full backtesting and validation protocol.

7. **Human Override:** A human operator can halt the system at any time with a single command. The system never locks out the human operator.

8. **No Overfitting:** All models are evaluated on out-of-sample data. In-sample performance is meaningless.

9. **Realistic Expectations:** The system aims for consistent 25%+ CAGR with < 12% drawdown. Any strategy claiming > 50% returns with < 5% drawdown is flagged as over-optimized and rejected.

10. **Continuous Improvement:** Weekly performance review is mandatory. Strategies that underperform for 2 consecutive months are suspended and re-evaluated.
